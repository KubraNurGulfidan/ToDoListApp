### ToDoListApp (C# / WinForms)

(TR)
## Proje Hakkında
Bu proje, günlük görevlerin yönetimini sağlamak amacıyla geliştirilmiş masaüstü tabanlı bir To-Do List uygulamasıdır. Kullanıcıların görev eklemesine, düzenlemesine, tamamlandı olarak işaretlemesine ve silmesine olanak tanır. Nesne Yönelimli Programlama (OOP) prensipleri kullanılarak C# dili ile geliştirilmiştir.

## Özellikler
- **Görev Yönetimi**: Yeni görevler oluşturma ve mevcut görevleri güncelleme.
- **Durum Takibi**: Görevlerin tamamlanma durumunu görsel olarak izleme.
- **Veri Listeleme**: Görevlerin düzenli bir şekilde görüntülenmesi ve yönetilmesi.

## Teknik Detaylar
- **Dil**: C#
- **Framework**: .NET / Windows Forms
- **Mimari**: Olay tabanlı programlama (Event-driven programming)

## About the Project
This project is a desktop-based To-Do List application developed to manage daily tasks. It allows users to add, edit, mark as completed, and delete tasks. Developed with C# using Object-Oriented Programming (OOP) principles.

## Features
- **Task Management**: Create new tasks and update existing ones.
- **Status Tracking**: Visually track the completion status of tasks.
- **Data Listing**: Organized display and management of tasks.

## Technical Details
- **Language**: C#
- **Framework**: .NET / Windows Forms
- **Architecture**: Event-driven programming
